FACT CHECK TOOLS

1. MONITORING MISINFORMATION TOOLS

USES: These tools help monitor and identify misinformation, fake news, and false narratives across digital platforms.

  1. Google Fact Check Explorer
  2. Tweetdeck/(X-Twitter)

2. FACIAL RECOGNITION TOOLS

USES: Facial recognition tools help identify individuals in images and verify identity claims in fact-checking investigations.

  1. Facecheck
  2. Pimeyes

3. ERROR LEVEL ANALYSIS TOOLS

USES: Error level analysis tools help detect image manipulation and editing by analyzing compression artifacts and inconsistencies.

  1. Forensically
  2. Fotoforensics

4. CHRONOLOCATION TOOLS

USES: Chronolocation tools help verify when images or videos were taken by analyzing shadows, sun position, and time-based data.

  1. Time and Date
  2. Shadow Map
  3. Suncalc

5. METADATA EXTRACTION TOOLS

USES: Metadata extraction tools help analyze hidden data in images and files to verify authenticity and gather investigative evidence.

  1. Exifeditor
  2. Exif.tool
  3. Metadata2go

6. RECOVER DELETED CONTENT

USES: Content recovery tools help access deleted or modified web content to track changes and verify historical information.

  1. Wayback Machine
  2. Archive.is
  3. CachedView
  4. Ghostarchive

7. AI CONTENT GENERATION TOOLS

USES: AI content generation tools help create test content and analyze patterns to better understand how AI-generated content appears.

  1. Gemini-2.5
  2. Gemini
  3. NotebookLM
  4. ChatGPT
  5. Perplexity AI

8. VISUAL VERIFICATION TOOLS

USES: Visual verification tools help authenticate images and videos, detect manipulation, and verify the authenticity of visual content.

  1. InVid Verification Plugin
  2. YouTube Data Viewer
  3. Google Lens
  4. Search by Image
  5. Sider-Chat

9. QUICK HELP TOOLS

USES: Quick help tools provide utilities for downloading, editing, and processing content for verification purposes.

  1. Facebook Video Downloader
  2. X Video Downloader
  3. Instagram Video Downloader
  4. YouTube Video Finder
  5. Remove Bg
  6. Cleanup Pictures
  7. Screen Recorder

10. GEOLOCATION TOOLS

USES: Geolocation tools help verify the location where images or videos were taken to confirm or debunk location-based claims.

  1. Google Earth Pro
  2. Google Maps
  3. Bing Maps
  4. OpenStreetMap
  5. Instant Street View
  6. Maps Search Tool
  7. Wiki Maps
  8. Picarta

11. SOCIAL MEDIA SEARCH TOOLS

USES: Social media search tools help track the origin and spread of content across different platforms for verification purposes.

  1. Graph
  2. Whopostedwhat
  3. FindMyFbID
  4. Webmii
  5. Mention
  6. MW Geofind
  7. Tofler
  8. Zaubacorp
  9. Instahunt

USES: AI detection tools help identify artificially generated content including deepfakes, AI-generated images, and synthetic media.

  1. Hive Moderation
  2. IsitAI?
  3. WasitAI?
  4. DEEPFAKE-O-METER
  5. SightengineAI
  6. Optic AI or NOT
  7. Deepfake Voice Detector
  8. ResembleAI
